Independent Variables

Consider a scenario where may be given two bonuses, A & B. The probability of us receiving the bonuses and the amount of the bonuses are given below.

A B
Amount 500 400
P 0.4 0.3

What is the variance of A and B?

Show Solution

The expected value and variance of A and B are as follows:

E(A) = 0.6(0) + 0.4(500) = 200
E(A²) = 0.6(0) + 0.4(250,000) = 100,000
Var(A) = 100,000 - (200)² = 60,000

E(B) = 0.7(0) + 0.3(400) = 120
E(B²) = 0.7(0) + 0.3(160,000) = 48,000
Var(B) = 48,000 - (120)² = 33,600

What would be the variance of the total amount of money we might receive?

Show Solution

To answer this question, we first need to find the probabilities of the different amount of money that we can receive. There are four possibilities:

A + B 0 400 500 900
P(A + B) 0.6x0.7 = 0.42 0.6x0.3 = 0.18 0.4x0.7 = 0.28 0.4x0.3 = 0.12

With this information, we can then find the expected value of A + B:

E(A + B) = 0.42(0) + 0.18(400) + 0.28(500) + 0.12(900) = 320

The variance of A + B is thus:

E((A+B)²) = 0.42(0) + 0.18(160,000) + 0.28(250,000) + 0.12(810,000) = 196,000
Var(A+B) = 196,000 - (320)² = 93,600

Notice how the result we got, 93,600, is equal to the sum of the individual variances of A and B. Therefore we can deduce the following formula, given that A and B are independent events:

Var(A + B) = Var(A) + Var(B)

Dependent Variables

Say we are given the following probability distribution table:

AND Y = 5 Y = 10
X = 10 0.3 0.2
X = 30 0.1 0.4

Are the variables X and Y independent? How do we know?

The two variables X and Y are independent only if:

P(X and Y) = P(X) x P(Y) holds for all values of X and Y.

But here, P(X and Y) when X = 30 and Y = 10 is 0.4. But:

When X = 30 and Y = 10,
P(X) x P(Y) = (0.1 + 0.4) x (0.2 + 0.4) = 0.5 x 0.6 = 0.30

Therefore X and Y are dependent.

Variance and Expected Value of Dependent Variables

Generally, if X and Y are dependent instead of independent, the formula for expected value always holds whereas the formula for variance does not.

That is, if X and Y are dependent, the following formula still holds:

E(A + B) = E(A) + E(B)

But the following would not hold:

Var(A + B) = Var(A) + Var(B)
