Tractor Problem: Revisited

A farm equipment dealer has obtained the joint probabilities of selling X tractors and Y combines on the same day:

AND Y = 0 Y = 1 Y = 2
X = 0 0.40 0.25 0.10 0.75
X = 1 0.20 0.05 0.00 0.25
0.60 0.30 0.10

Suppose that each tractor has a profit of $600 and each combine has a profit of $200.

Determine the expected daily profit.

Show Solution

The expected value for X and Y are as follows:

E(X) = 0(0.75) + 1(0.25) = 0.25
E(Y) = 0(0.60) + 1(0.30) + 2(0.10) = 0.50

Therefore the expected daily profit would be:

E(Profit) =
600E(X) + 200E(Y) = 600(0.25) + 200(0.50) = $250

Determine the variance of daily profit.

Show Solution

Firstly, we need to find the probability distribution of the profit. There are 2 x 3 = 6 possible combinations of X and Y and we need to account for the total profit for each combination:

X Y Profit P(Profit)
0 0 0 0.40
0 1 200 0.25
0 2 400 0.10
1 0 600 0.20
1 1 800 0.05
1 2 1000 0.00

Using this table, we can easily find the variance of X and Y. Since we know that E(Profit) = 250, then the variance of the profit is:

Var(Profit) = (0-250)²(0.4) + (200-250)²(0.25) + (400-250)²(0.10) + (600-250)²(0.20) + (800-250)²(0.05) + (1000-250)²(0.00) = 67,500
