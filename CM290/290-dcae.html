Decision Analysis Example

Sequential Decisions Problem

An investor intends to purchase a piece of real estate. She has to choose between buying an apartment, a warehouse, or a retail space. The profit made will depend on future economic decisions, which is either good or lame. The payoff matrix is as follows:

Profits Good / G Lame / L
Apartment / A 55 10
Warehouse / W 30 20
Retail / R 100 -40

What is the best decision using the minimax regret criterion?

The regret matrix would be:

Regret Good / G Lame / L
Apartment / A 45 10
Warehouse / W 70 0
Retail / R 0 60

Therefore the best decision is A because it has the lowest maximum regret of 45, compared to the 70 of W and the 60 of R.

Now say we have information on the probility of the economy being either good or lame:

Good Lame
P() 0.6 0.4

What is the best decision using expected monetary value?

The best decision is simply the one with the highest expected value. Therefore:

Expected Monetary Value
A 0.6(55) + 0.4(10) = 37
W 0.6(30) + 0.4(20) = 26
R 0.6(100) + 0.4(-40) = 44

Since R has the highest expected value, R would be the best decision.

What is the expected value of perfect information on future economic conditions?

We need to know what the expected profit with perfect information (EPPI) is. To find this, we first find the decisions that would result in the highest output given the economic conditions.

Profits Good / G Lame / L
Apartment / A 55 10
Warehouse / W 30 20
Retail / R 100 -40

If the economy is good, the best decision is retail with a profit of 100. If the economy is lame, the best decision would be warehouse, with a profit of 20. These outcomes are shown highlighted above.

The EPPI would just be the expected value given these two values:

EPPI = 0.6(100) + 0.4(20) = 68

The value of this information would be the difference between the EPPI and the EMV we found earlier:

Value of information = 68 - 44 = 24

Sequential Decisions

The investor may hire a professional economist who can forecast the economy to be either booming, coasting, or declining. An examination of previous forecasts shows the following probabilities:

IF Good Lame
Booming 0.30 0.05
Coasting 0.50 0.50
Declining 0.20 0.45

How much would the investor pay for the economist?

Essentially we need to find the expected value considering the information that is given by the economist. To do that, we need to know:

  • What is the probability of the economy being good given that the economist says the economy is booming?
  • What if the economist says the economy is coasting?
  • What if the economist says the economy is declining?

The first point is the same as the probability P(G | B). This is equivalent to:

P(G | B) = P(GB) / P(B)

Therefore we need to find P(GB) and P(B).

  • P(GB) is simply P(G) multiplied by P(B | G).
  • P(B) can be found by summing P(GB) and P(LB):

P(B) = P(GB) + P(LB) = 0.6(0.3) + 0.4(0.05) = 0.20

P(G | B) is then:

P(G | B) = 0.18 / 0.20 = 0.90


The second point is the same as the probability P(G | C). Following the same process:

P(GC) = 0.6(0.5) = 0.3

P(C) = P(GC) + P(LC) = 0.6(0.5) + 0.4(0.5) = 0.5

P(G | C) = 0.3 / 0.5 = 0.6


Subsequently, we can find that the third point, P(G | D), is equal to 0.4.

We then need to find the expected value of each decision considering the forecast of the economist. If the economist says the economy is booming, the expected value of the decisions would be:

Booming
A 0.9(55) + (1-0.9)(10) = 50.5
W 0.9(30) + 0.1(20) = 29
R 0.9(100) + 0.1(-40) = 86

Doing this for coastal and declining would result in the following table:

Booming Coastal Declining
A 0.9(55) + (1-0.9)(10) = 50.5 0.6(55) + (1-0.6)(10) = 37 0.4(55) + (1-0.4)(10) = 28
W 0.9(30) + 0.1(20) = 29 26 24
R 0.9(100) + 0.1(-40) = 86 44 16

Knowing all of these expected values, we would only choose the decisions with the highest EVs. So if the economist's report says:

  • Booming - pick R
  • Coastal - pick R
  • Declining - pick A

These decisions and their EVs are reflected as the selceted boxes in the above table.

Since we calculated the probabilities of booming, coastal, and declining before (shown below), we can also find the expected profits if we decide to hire the economist:

Booming Coasting Declining
P() 0.20 0.50 0.30

E(X) = 0.2(86) + 0.5(44) + 0.3(28) = 47.6

Since previous analysis shows that, without the economist, the expected value is 44, the value of information given by the economist is:

Expected Value of Sample Information (EVSI) = 47.6 - 44.0 = 3.6

This EVSI is the maximum amount the investor would be willing to pay the economist.

What is the efficiency of this sample information?

The efficiency is defined as the ratio between the EVSI and EVPI:

Efficiency = 3.6 / 24 = 15%
