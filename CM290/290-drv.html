Discrete Random Variables

What is a discrete random variable?

A random variable is a variable whose value is a numerical outcome of a random event. A discrete random variable is a random variable whose possible values are countable.

For example, the random variable X that represents the outcome of a thrown fair dice is discrete because the outcome can only be one of: [1, 2, 3, 4, 5, 6].

Expected Value

The expected value for a discrete random variable is given as follows:

E(X) = ∑xp(x)

Consider the following example for a random variable with the following probability distribution:

X 10 11 12 13 14
P(X) 0.4 0.2 0.2 0.1 0.1

The expected value for this random variable would thus be:

E(X) = 10(0.4) + 11(0.2) + 12(0.2) + 13(0.1) + 14(0.1) = 11.3

Example: Money Game

Consider a game where you pick a bill out of the box and get to keep it. The box contains 1 $20 bill, 2 $14 bills, and 3 $4 bills. What would be the long run average payout?

Solution

This problem would have the following probability distribution:

X 4 14 20
P(X) 3/6 2/6 1/6

The "long run average payout" is the same as asking for the expected value, therefore:

E(X) = 4(3/6) + 14(2/6) + 20(1/6) = 10

Example II: Money Game with Replacement

This variation of the game allows you to pick two bills - with replacement - and keep. The question still stands - what would be the long-run average payout?

Solution

This problem would have the following probability distribution:

X 8 18 24 28 34 40
P(X) 9/36 12/36 6/36 4/36 4/36 1/36

The "long run average payout" is the same as asking for the expected value, therefore:

E(X) = 8(9/36) + 18(12/36) + 24(6/36) + 28(4/36) + 34(4/36) + 40(1/36) = 20

Variance & Standard Deviation

Variance is the average squared deviation of values from the mean. Standard deviation is the square root of the variance, or intuitively, the average deviation of values from the mean.

The variance of a random variable X can be calculated as follows:

σ²(X) = Σ[x-E(X)]²p(x)

Calculating the standard deviation is simply taking the square root of the variance.

Example

Consider a random variable X that represents the $ won in a single game. The probability distribution is as follows:

X 0 1 19 20
P(X) 1/4 1/4 1/4 1/4

What is the variance and standard deviation of X?

Solution

To get the variance, we need the deviation from E(X) squared. Since the probability is all the same, the expected value would be:

E(X) = 1/4(0 + 1 + 19 + 20) = 10

Therefore the deviation from E(X) squared would be:

X 0 1 19 20
P(X) 1/4 1/4 1/4 1/4
[1-E(X)]² 100 81 81 100

Therefore the variance of X would be:

Var(X) = 1/4(100 + 81 + 81 + 100) = 90.5

More Expected Value and Variance

Below is another formula that we can use to find the variance of X:

Var(X) = E(X²) - E(X)²

The expected value of a multiple of X is simply the multiple of x times E(X). That is:

E(aX) = aE(X)

On the other hand, the variance of a multiple of X is equal to the multiple squared times the variance of X. That is:

σ²(aX) = a²σ²(X)
