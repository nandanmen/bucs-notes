Question 1

Active Camera Co. wants to develop a cost formula to estimate the fixed and variable components of the company's monthly manufacturing overhead costs.

Develop an MOH cost formula and estimate the overhead cost at 22,500 MH.

Show Solution

The high value is in February 2013, with machine hours of 25,000 hours and overhead costs of $99,000.

The low value is in August 2013, with machine hours of 10,000 hours and overhead costs of $64,500.

Total variable cost is:

(99,000 - 64,500) / (25,000 - 10,000) =
34,500/15,000 = $2.3 / MH

Total fixed cost is:

99,000 - (2.3 x 25,000) = $41,500

Therefore the cost formula is:

41,500 + 2.3MH

Using this formula, the total cost at 22,500 machine hours will be:

41,500 + 2.3(22,500) = $93,250

You learn that ACC leases all of its manufacturing equipment. The lease calls for a flat monthly fee up to 19,500 MH. If the lease exceeds 19,500, the fee becomes strictly variable with respect to the total number of machine hours used during the month.

Explain any concerns you might have with the high-low model that you devised.

Show Solution

Our high-low model uses a "low" value of 10,000 machine hours. With this new information, we know that at this level, costs are still fixed. As a result, the high-low method will provide an inaccurate value as long as the "low" value is anything less than 19,500 machine hours.

Given the new information, estimate the overhead cost at 22,500 machine hours.

Show Solution

We need to pick a new "low" that is higher than the 19,500 machine hours. The lowest data point for this is in May 2014, with 20,000 machine hours and $80,000 in total costs. Running the high low method again we get:

(99,000 - 80,000)/(25,000 - 20,000) =
19,000/5,000 = $3.8 / MH

Fixed cost = 80,000 - 3.8(20,000) = $4,000

The overhead cost for 22,500 machine hours using this model would be:

4,000 + 3.8(22,500) = $89,500

Provide an estimate for the flat monthly fee for the manufacturing equipment.

Show Solution

For this we need to use the high-low method for the range of machine hours below 19,500. The high point is at 19,000 machine hours at $78,000, and the low point is 10,000 hours at $64,500.

(78,000 - 64,500) / (19,000 - 10,000) =
13,500/9,000 = $1.5 / MH

The flat monthly fee would be the fixed cost portion minus the fixed cost for the non-lease portion (range above 19,500). Therefore:

64,500 - 1.5(10,000) = 49,500 - 4,000 = $45,500
